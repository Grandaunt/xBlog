1、服务器接受数据。支持此xml、json、x-www-form-urlencoded三种格式.

例： http://www.huxuemin.com/api/articles.xml 、http://www.huxuemin.com/api/articles.json

1、获取文章目录: /api/article。请求类型：GET。

2、文章内容：/api/article/{articleId}。请求类型：GET。
例：/api/article/11804505069905。

3、删除文章：/api/article/{articleId}。请求类型：DELETE。

4、关闭文章评论：/api/article/{articleId}/discuss/close。请求类型：PUT。

5、打开文章评论：/api/article/{articleId}/discuss/open。请求类型：PUT。

6、查询文章是否可以评论：/api/article/{articleId}/discuss/canreply。请求类型：GET。

7、发布新文章：/api/article。请求类型：POST。

8、修改文章：/api/article/{articleId}。请求类型：PUT。

1、文章所有评论：/api/article/{articleId}/discuss。请求类型：GET。

2、评论页面数：/api/article/{articleId}/discuss/page，每页10条评论。请求类型：GET。

3、获取指定页评论：/api/article/{articleId}/discuss/page/{number}，请求类型：GET。

4、回复文章：/api/article/{articleId}/discuss，请求类型：POST。

5、回复文章评论：/api/article/{articleId}/discuss/{discussId}，请求类型：POST。

1、用户登录信息：/api/login，请求类型：GET。

2、用户登录：/api/login，请求类型：POST。

3、用户退出：/api/login，请求类型：DELETE。

4、是否具有修改该用户资料权限：/api/user/{username}/canedit，请求类型：GET。

5、获取用户资料：/api/user/{username}，请求类型：GET。

6、修改该用户资料：/api/user/{username}，请求类型：PUT。

7、用户注册：/api/user，请求类型：POST。
