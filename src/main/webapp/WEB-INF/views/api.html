文章目录: /api/articles。 支持方法：get:获取文章目录。

文章：/api/articles/{articleId}。例：/api/articles/11804505069905。 支持方法：get:获取文章。

评论：/api/articles/{articleId}/discuss/{discussId}。

评论页面数：/api/articles/{articleId}/discuss/page。每页10条评论。

获取一页评论：/api/articles/{articleId}/discuss/page／{number}。每页10条评论。

用户登录信息：/api/login

用户信息：/api/user/{username}。

判断是否具有修改用户资料权限：/api/user/{username}/canedit
